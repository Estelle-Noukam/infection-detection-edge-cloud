# IoT Edge–Cloud Infection Detection

Projet académique de détection précoce d'infection chez des patients à domicile utilisant une architecture IoT Edge–Cloud.

## Architecture

Edge : Raspberry Pi 4  
Cloud : API Flask sur Kali Linux

## Paramètres surveillés

- température
- fréquence cardiaque
- saturation SpO2
- fréquence respiratoire
- pression artérielle
- état de confusion

## Technologies

- Python
- Flask
- REST API
- Raspberry Pi
